Shows a slideshow of .bmp pictures.

Pics need to be numbered 01 (to 99) max, the figures
can be anywhere in the filename but the rest of the
filename should be the same for all pics. EG:

PIC(01).bmp
PIC(02).bmp
PIC(03).bmp

Put .bmp file in PICS subdirectory.

Pics should be Windows format .bmp, uncompressed, in 256 colours,
maximum 368x256 with the width a multiple of 8 pixels.
