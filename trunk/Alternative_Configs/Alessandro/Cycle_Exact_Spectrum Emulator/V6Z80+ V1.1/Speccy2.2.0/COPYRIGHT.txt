2. ZX SPECTRUM ROM
==================

This core uses the original ZX Spectrum 48/128K ROM, distributed with the
binary package.

Amstrad, the current owners of the Sinclair ZX Spectrum, allow free
distribution of the ZX Spectrum 48K/128K ROMs; see

http://www.worldofspectrum.org/permits/amstrad-roms.txt

for further details.

Amstrad have kindly given their permission for the redistribution of
their copyrighted material but retain that copyright. Thank you Amstrad.

================

This core makes use of the YM-2149 VHDL code developed by Mike Johnson.
That is a simulation model of YM2149 (AY-3-8910 with bells on)
Copyright (c) MikeJ - Jan 2005 - All rights reserved.

Many thanks to Mike Johnson for this work!
