Textedit - a simple text editor for FLOS by Phil
------------------------------------------------

Command line use:
-----------------

Textedit [optional filename]


Controls:
---------

ALT+Esc - Prompts: Save, Quit or Cancel.

ALT+L - Load a text file

ALT+S - Save current text file (and return to editor)


Limitations
-----------

Text file must be smaller than about 37KB
